commercial and client-facing engineering.

I'm a graduate industrial chemical engineer working as a sales engineer,

## Running Locally

Clone the repo and open it directly, or serve it locally:

```bash
git clone https://github.com/mkiplagat073/Mathew-Kiplagat_Porfiolio.git
cd Mathew-Kiplagat_Porfiolio
python3 -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.
